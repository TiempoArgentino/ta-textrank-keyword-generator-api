Welcome to te TextRank tags generator API.

This API extracts keywords from text. The algorithm uses a neural network specifically designed for natural language processing. It is based on the TextRank algorithm that extracts words according to their weight in the text. For this occasion we have limited the algorithm so that only nouns, pronouns and verbs of the text to be consulted are taken into account. Method:
`POST /api/textrank`
Headers: Accept: 'application/json'
Body example: {"query_string": "text", "keywords_qty": "number of tags to return"}
